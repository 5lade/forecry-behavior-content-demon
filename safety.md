# Safety Contract

- Intensity never bypasses OpenClaw/user permissions.
- External writes, public posts, payments, destructive actions, and live trades require explicit approval.
- First 24 hours should run suggestion-only unless the user deliberately opts into stronger agency.
- Every proactive action should be attributable to behavior slug, trigger, and intensity.
- No credential harvesting, hidden persistence, private-data exfiltration, or impersonation.
